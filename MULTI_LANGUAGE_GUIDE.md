# Multi-language Support Implementation Guide

## 🌍 Overview

This branch demonstrates multi-language support for the Smart Calculator project with:
- **Languages**: English (en), Brazilian Portuguese (br), Polish (pl), German (de)
- **SEO-Optimized**: Manual meta titles, descriptions, and slugs
- **Auto-translated Content**: FREE using Google Gemini API
- **Zero Cost**: Uses free Gemini API tier

## 📁 File Structure

```
app/content/
├── meta/
│   └── calculators.ts          # ✍️ MANUAL: SEO meta, slugs, page titles
│
└── calculators/
    └── bmi-calculator/
        ├── en.json             # ✍️ You write English content
        ├── br.json             # 🤖 AUTO: Generated by script
        ├── pl.json             # 🤖 AUTO: Generated by script
        └── de.json             # 🤖 AUTO: Generated by script

lib/translation/
└── gemini-translate.ts         # Translation utility (FREE API)

scripts/
└── translate-content.ts        # Auto-translation script
```

## 🚀 Quick Start

### Step 1: Get FREE Gemini API Key

1. Visit: https://makersuite.google.com/app/apikey
2. Sign in with Google account
3. Click "Create API Key"
4. Copy the key

### Step 2: Setup Environment

```bash
# Copy example env file
cp .env.local.example .env.local

# Add your Gemini API key
# Edit .env.local and replace with your actual key
GEMINI_API_KEY=your_actual_api_key_here
```

### Step 3: Run Translation

```bash
# Install dependencies (already done)
npm install

# Run translation script
npx tsx scripts/translate-content.ts
```

This will:
- ✅ Read `en.json` files
- ✅ Auto-translate to `br.json`, `pl.json`, `de.json`
- ✅ Save translated files
- ✅ All for FREE!

## 📝 What's Manual vs Auto?

### ✍️ Manual (You write for SEO):

**File**: `app/content/meta/calculators.ts`

```typescript
export const calculatorsMeta = {
  "bmi-calculator": {
    en: {
      metaTitle: "BMI Calculator - Free Body Mass Index Calculator Online",
      metaDescription: "Calculate your BMI...",
      slug: "bmi-calculator",
      pageTitle: "BMI Calculator",
    },
    br: {
      metaTitle: "Calculadora de IMC - Calcule seu IMC Grátis",
      metaDescription: "Calcule seu IMC...",
      slug: "calculadora-imc",
      pageTitle: "Calculadora de IMC",
    },
    // ... pl, de
  }
};
```

**Why Manual?**
- Better SEO rankings
- Keyword optimization
- Clean, localized URLs

### 🤖 Auto-translated (Script handles):

**File**: `app/content/calculators/bmi-calculator/en.json`

```json
{
  "sections": [
    {
      "heading": "What is a BMI Calculator?",
      "content": "A BMI Calculator is a health tool..."
    }
  ],
  "faq": [...]
}
```

**Why Auto?**
- Saves time
- Consistent translations
- Easy to update
- FREE!
